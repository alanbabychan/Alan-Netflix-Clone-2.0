# 🎬 Alan-Netflix Clone 2.0

> A fully responsive **Netflix-inspired video streaming platform** built using **React.js, TypeScript, and modern frontend technologies**.

---

## 🚀 Overview

**Alan-Netflix Clone 2.0** is an advanced upgrade of the original Netflix Clone project, designed to closely replicate the **real Netflix user experience**.

The application allows users to browse movies, search titles, view trailers, manage profiles, and stream content using live movie data from **TMDb (The Movie Database) API**.

This version focuses on:

* ⚡ Performance optimization
* 🎨 Modern UI/UX
* 🔐 Authentication improvements
* 📱 Fully responsive layout
* 🎥 Real streaming behavior

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/alanbabychan/Alan-Netflix-Clone-2.0.git
cd Alan-Netflix-Clone-2.0
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Application runs at:

```
http://localhost:5173
```
